## 🚀 新增功能 / New Features

### 远程文件管理模块 / Remote Files Management Module
- ✨ **新增"远程文件"模块** - 在侧边栏新增远程文件入口 / Added "Remote Files" module in sidebar
- ✨ **云端文件浏览器** - 支持浏览 OSS 存储桶中的文件和文件夹 / Cloud file browser - browse files and folders in OSS buckets
- ✨ **文件上传** - 支持将本地文件直接上传到云端 / File upload - upload local files directly to cloud
- ✨ **文件下载** - 支持将云端文件下载到本地指定路径 / File download - download cloud files to local specified path
- ✨ **文件删除** - 支持删除云端文件（带确认提示）/ File deletion - delete cloud files (with confirmation)
- ✨ **目录导航** - 支持在云端文件夹中导航切换 / Directory navigation - navigate between cloud folders
